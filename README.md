# Prompt Engineering Task 2 – AI Content Marketing using UGC Ads

## About This Project

As part of my Prompt Engineering Internship at Future Interns, I worked on creating AI-generated marketing content using effective prompts.

For this project, I selected a reusable water bottle brand called EcoSip. The aim was to generate UGC-style advertisement content such as hooks, short scripts, and call-to-actions using AI.

Through this task, I explored how different prompt styles can influence the quality and creativity of AI-generated outputs.

## Objectives

- Understand the basics of Prompt Engineering
- Create prompts for marketing content generation
- Generate UGC-style advertisements
- Compare different prompt outputs
- Learn how AI can support content creation

## Product Chosen

**EcoSip Reusable Water Bottle**

## Target Audience

- College Students
- Young Adults
- Daily Commuters

## Files Included

- README.md
- prompts.txt
- generated_hooks.txt
- ugc_ad_scripts.txt
- prompt_comparison.txt
- report.docx
- screenshots

## What I Learned

While working on this project, I learned that the quality of AI-generated content depends heavily on how clearly the prompt is written.

I also observed that adding details such as audience, tone, and content structure helped generate more relevant and natural outputs.

This task gave me practical experience in using Prompt Engineering for content marketing and helped me understand its real-world applications.
